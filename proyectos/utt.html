02.

Algoritmo de Persistencia


// Lógica de Persistencia y Búsqueda
if (ahora - tiempoUltimoDetectado < persistencia) {
    if (ultimoLado == 1) girarIzquierda(velGiro);
    else girarDerecha(velGiro);
} else {
    buscarOponente(); // Máquina de estados 4 fases
}

Implementé una máquina de estados de 4 fases para el patrón de búsqueda y una variable de persistencia. Esto permite que el robot mantenga el objetivo bloqueado aunque el sensor pierda la señal momentáneamente por la velocidad del choque.
