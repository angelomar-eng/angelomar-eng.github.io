04.

Lógica de Control con Bluepad32


// Mapeo de control R2/L2 y D-pad
void loop() {
    BP32.update();
    ControllerPtr myGamepad = myControllers[0];

    if (myGamepad && myGamepad->isConnected()) {
        int r2 = map(myGamepad->throttle(), 0, 740, 0, 1023); 
        int l2 = map(myGamepad->brake(), 0, 900, 0, 1023); 
        
        if (myGamepad->dpad() == 4) driveMotor(2, 900);  
        else if (r2 > 50) driveMotor(1, r2);
        else if (l2 > 50) driveMotor(-1, l2);
        else driveMotor(0, 0);
    }
}

Se implementó un sistema basado en Bluetooth HID. La resolución de 10 bits en el PWM permite una aceleración progresiva y precisa, vital para el empuje constante.

  • Frecuencia PWM: 20,000 Hz
  • Mapeo: 0-1023 (10-bit)
  • Latencia: < 10ms
